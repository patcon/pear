Drush's PEAR Channel Server

Using this Channel

This channel is to be used with the PEAR installer.

Registering the channel: 
pear channel-discover drush-ops.github.com/pear
Listing available packages: 
pear remote-list -c drush-ops
Installing a package: 
pear install drush-ops/package_name
Installing a specific version/stability: 
pear install drush-ops/package_name-1.0.0
pear install drush-ops/package_name-beta
Receiving updates via a feed: 
http://drush-ops.github.com/pear/feed.xml
